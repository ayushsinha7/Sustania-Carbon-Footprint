Sustaina: Real-Time Digital Twin & Predictive Carbon Footprint Simulator

Sustaina is an advanced, responsive interactive dashboard designed to model localized environmental impacts, track individual carbon habits, and dynamically project ecosystem stabilization thresholds. Built as an analytical tool for the Hack2Skill challenge, the simulator visualizes active habit metrics against industrial baselines to offer clear mitigation pathways

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

   🛠️ Key Architectural Features
* **Dynamic Simulation Engine:** Computes complex mathematical real-time carbon offsets utilizing regional Indian grid coefficients and global transit metrics.
* **Responsive Performance Metrics:** Tracks air quality percentages, direct power draw allocations, and annual tree neutralization thresholds simultaneously.
* **Interactive Metric Tuning:** Provides configurable parameters across three distinct lifecycle vectors: Utility Footprints, Transit Logistics, and Nutritional Allocation Models.
* **Sleek Glassmorphic Dark UI:** Formatted with modern responsive viewport structures built for immediate clarity and analytical focus.
